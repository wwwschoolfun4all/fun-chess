# Fun Chess Multiplayer

A modern, interactive chess game with power-ups, multiplayer functionality, and extensive customization options.

## Features

- **34+ Power-ups**: Bomb, Freeze, Shield, Lightning, and more
- **Multiplayer Modes**: Local, AI, and Online multiplayer
- **Customization**: 40+ board themes, 30+ piece skins, 30+ special effects
- **Smart AI**: Three difficulty levels with minimax algorithm
- **Real-time Multiplayer**: Global matchmaking and private rooms
